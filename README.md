# Bike Rental Demand Prediction

## Project Overview
This project predicts the number of bikes rented based on factors like temperature, weather, humidity, and season.

## Dataset
The dataset contains information about:
- Temperature
- Humidity
- Wind speed
- Season
- Holiday
- Working day

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Models
- Linear Regression
- Random Forest
- Gradient Boosting

## Evaluation Metrics
- RMSE
- R² Score

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Training
5. Model Evaluation

## Conclusion
Random Forest performed better compared to other models for predicting bike rental demand.
